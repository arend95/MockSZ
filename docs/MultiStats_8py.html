Function to be called in multiprocessing.

Calculates chunks of the scattering, as function of s, over beta. Uses the Maxwell-Juttner distribution for the electron population.

Parameters

beta_limArray with lower limits on beta, as function of s.

Teelectron temperature, in Kelvin.
Returns
